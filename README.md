# Video Downloader Bot

Бот для особистого використання: завантажує відео з TikTok, Instagram, YouTube тощо без водяних знаків.

## Як запустити
1. `pip install -r requirements.txt`
2. Створи `.env` з `BOT_TOKEN=...`
3. `python main.py`
